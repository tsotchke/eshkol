### `BorrowChecker`

*class* ‚Äî line 230

```c
class BorrowChecker
```

Borrow checker for ownership and borrowing semantics Tracks: - Ownership of values (owned, moved, dropped) - Active borrows (shared or mutable) - Borrow lifetimes (scopes) Rules enforced: 1. A value can be moved only once 2. While borrowed, a value cannot be moved 3. Mutable borrows are exclusive (no other borrows allowed) 4. Shared borrows allow multiple readers 5. Borrows must not outlive the borrowed value
