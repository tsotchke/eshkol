### `SigmaType`

*Struct* — line 293

```c
struct SigmaType
```

Dependent pair type (Σ-type / existential type) A Σ-type represents an existential quantification: Σ(x:A).B(x) meaning "there exists an x of type A such that B(x) holds". Examples: - Σ(n:Nat).Vector<Float64,n> - a vector of some length - Σ(T:Type).List<T> - a list of some element type In HoTT, Σ-types are dual to Π-types: - Π-type: "for all x:A, B(x)" (universal) - Σ-type: "exists x:A, B(x)" (existential) A Σ-type value is a dependent pair (a, b) where: - a : A (the witness) - b : B(a) (the dependent component)
