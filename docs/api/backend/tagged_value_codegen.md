### `TaggedValueCodegen`

*class* — line 50

```c
class TaggedValueCodegen
```

TaggedValueCodegen handles packing/unpacking of tagged values. Tagged values are the runtime representation of all Eshkol values. This class provides methods to: - Pack primitive values (int64, double, bool, char, null) into tagged form - Pack pointer values (cons, vector, tensor, string, closure) into tagged form - Unpack tagged values back to primitive LLVM types - Query type tags and flags from tagged values - Safely extract i64 from various value types

Ensure a raw LLVM value is packed as a tagged_value struct.
