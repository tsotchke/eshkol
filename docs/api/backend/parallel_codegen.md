### `ParallelCodegen`

*class* — line 52

```c
class ParallelCodegen
```

ParallelCodegen generates LLVM IR for parallel operations. The parallel primitives call into C runtime functions that use a thread pool for parallel execution. Each worker thread has a thread-local arena for memory allocation. Usage in Scheme: (parallel-map fn list) -> Apply fn to each element in parallel (parallel-fold fn init list) -> Reduce list with fn (map phase parallel) (parallel-filter pred list) -> Keep elements where pred returns true (parallel-for-each fn list) -> Apply fn for side effects (thread-pool-info) -> Get number of worker threads
