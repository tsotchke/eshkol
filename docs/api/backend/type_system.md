### `TypeSystem`

*class* â€” line 35

```c
class TypeSystem
```

TypeSystem manages all LLVM types used in the Eshkol compiler. It provides: - Cached primitive types (int64, double, ptr, etc.) - Struct types for runtime values (tagged_value, dual_number, ad_node, tensor) - Type creation helpers Usage: TypeSystem types(context); auto int64_t = types.getInt64Type(); auto tagged = types.getTaggedValueType();
