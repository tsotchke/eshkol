### `AutodiffCodegen`

*class* ‚Äî line 41

```c
class AutodiffCodegen
```

AutodiffCodegen handles automatic differentiation operations. Eshkol supports two modes of automatic differentiation: 1. Forward-mode AD using dual numbers (efficient for few inputs, many outputs) 2. Reverse-mode AD using tape recording (efficient for many inputs, few outputs)
