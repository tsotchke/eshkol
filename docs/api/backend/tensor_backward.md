### `eshkol_tensor_backward_dispatch`

*Function* — line 358

```c
void eshkol_tensor_backward_dispatch(void* ad_node_ptr);
```

Main entry point called from LLVM-generated backward pass. Reads the AD node's type, tensor_gradient (upstream grad), saved_tensors, params, shape/ndim, then dispatches to the appropriate eshkol_backward_* function and accumulates gradients on input nodes. Convention for params (stored as 6 x int64_t): Conv2d: [kernel_h, kernel_w, stride_h, stride_w, channels_in, channels_out] MaxPool2d: [kernel_h, kernel_w, stride_h, stride_w, channels, batch_size] AvgPool2d: [kernel_h, kernel_w, stride_h, stride_w, channels, batch_size] BatchNorm: [batch_size, feature_size, 0, 0, 0, 0] LayerNorm: [num_samples, feature_size, 0, 0, 0, 0] Matmul: [M, K, N, 0, 0, 0] Transpose: [rows, cols, 0, 0, 0, 0] Reshape: [total_elements, 0, 0, 0, 0, 0] Sum: [input_total_elements, 0, 0, 0, 0, 0] Mean: [input_total_elements, 0, 0, 0, 0, 0] Attention: [seq_q, seq_k, d_k, d_v, scale_bits, 0] MHA: [seq_q, seq_k, d_model, num_heads, 0, 0] PosEnc: [total_elements, 0, 0, 0, 0, 0] Embedding: [num_indices, d_model, vocab_size, 0, 0, 0]

**Parameters**

- `ad_node_ptr` — AD node whose backward pass should be dispatched (as eshkol_ad_node_t*, opaque here)
