### `LogicWorkspaceCodegen`

*class* ‚Äî line 48

```c
class LogicWorkspaceCodegen
```

LogicWorkspaceCodegen generates LLVM IR for Eshkol's consciousness-engine primitives. Construction is lightweight; the heavy work (runtime function declarations) happens lazily on first use via getOrDeclareRuntimeFunc(). Several methods recurse into AST evaluation. Wire that callback via setCodegenASTCallback() ‚Äî the parent EshkolLLVMCodeGen passes a wrapper that bounces back into its own codegenAST().
