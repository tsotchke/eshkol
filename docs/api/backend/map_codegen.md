### `MapCodegen`

*class* — line 40

```c
class MapCodegen
```

MapCodegen handles higher-order list mapping operations. Key responsibilities: - Resolve procedure argument (lambda, closure, builtin) - Apply procedure to each element of list(s) - Build result list in arena memory - Handle closure captures during mapping
