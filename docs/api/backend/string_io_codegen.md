### `StringIOCodegen::stringByteLength`

*Function* ‚Äî line 83

```c
llvm::Value* stringByteLength(const eshkol_operations_t* op);
```

Get string BYTE length (header.size, no UTF-8 decoding): (string-byte-length s). Distinct from stringLength(), which counts codepoints ‚Äî a multibyte-UTF-8 string (e.g. containing an em-dash, 3 bytes / 1 char) has a byte length > its char length. Exists so FFI calls like `(fwrite s 1 (string-byte-length s) f)` write the whole string instead of truncating to the codepoint count.

**Parameters**
