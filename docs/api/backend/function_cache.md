### `FunctionCache`

*class* — line 35

```c
class FunctionCache
```

FunctionCache provides lazy-loaded C library function declarations. Each function is declared in the LLVM module only when first requested, then cached for subsequent calls. This avoids cluttering the module with unused function declarations. Usage: FunctionCache funcs(module, types); auto malloc_fn = funcs.getMalloc(); auto strlen_fn = funcs.getStrlen();
