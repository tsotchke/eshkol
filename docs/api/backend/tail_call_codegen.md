### `TailCallCodegen`

*class* — line 44

```c
class TailCallCodegen
```

TailCallCodegen handles tail call optimization. LLVM TCO Requirements: 1. Call must be immediately followed by a return 2. Caller and callee must have compatible calling conventions 3. The return value must be the result of the call (or void) 4. No instructions between call and return (except bitcasts) For closures with captured environments, we use a trampoline approach: - Functions return either a result or a "bounce" (thunk to continue) - The trampoline loop evaluates bounces until a result is reached
