# `eshkol/core/image_io.h`

Image I/O API — read/write PNG, JPEG, BMP as tensor data. Images are normalized doubles (0.0-1.0). Shapes: (H,W,C) for color, (H,W) for grayscale. Memory ownership: returned arrays are arena-allocated from the Eshkol global arena. Callers must **not** free() them — the arena releases all allocations together at arena_reset() / arena_destroy() time. Copyright (C) Tsotchke Corporation. MIT License.

Read image file → flat double array (normalized 0-1, row-major, HWC layout).

**Parameters**

- `out_w` — Output: image width
- `out_h` — Output: image height
- `out_c` — Output: number of channels (1=gray, 3=RGB, 4=RGBA)

**Returns**
