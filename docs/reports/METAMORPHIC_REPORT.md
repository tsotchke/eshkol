# P7c Metamorphic/property Harness Report

Date: 2026-07-04
Branch: `test/p7c-metamorphic`

## Scope

The P7c harness adds deterministic metamorphic/property tests for Eshkol. Test
data is generated by `tests/metamorphic/gen_metamorphic.py` using an explicit
xorshift64* PRNG seeded by case index. Generated `.esk` probes contain no
runtime randomness and are committed under `tests/metamorphic/generated/`.

Default corpus: 32 trials per generated family.
