# Depth-parametric AD oracle report (P6a)

Generated by `scripts/run_ad_depth.sh` (`scripts/gen_ad_depth.py` + `scripts/ad_depth_report.py`).

Every AD value is swept at nesting depth **d = 1..8** and checked against a **closed-form analytic** ground truth (n-th derivative of the base shape), corroborated at shallow depth by an in-language n-th central-difference stencil. `P`=PASS `F`=FAIL(silent wrong) `L`=LIMIT(clean crash/limit).

**MCD** = max-correct-depth (largest d with 1..d all PASS, worst lane). Lanes: jit, aot.
